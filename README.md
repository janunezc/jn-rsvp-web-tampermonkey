# jn-rsvp-web-tampermonkey
Rapid Serial Visual Presentation: This TamperMonkey script bolds the first two syllables (sort of) of every word in every HTML paragraph to improve readability for neurodiverse conditions.

# Warning
This tampermonkey is provided as-is with the best intentions to help. We accept no liability on it.

Remember to modify the @match tag in the script to have your web site covered by it.

```
// @match        https://*.costaricamakers.com/*
// @match        https://*.nunez-technologies.com/*
// @match        https://*.c9eb.space/*
```
